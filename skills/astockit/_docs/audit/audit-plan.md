## Audit Methodology

For each skill, I will assess:

1. **Workflow alignment:** Does it map clearly to quantitative workflow stages?
2. **Technical completeness:** Are all technical specifications detailed enough?
3. **Quality gates:** Are validation checkpoints explicit?
4. **A-share specifics:** Are China A-share constraints fully addressed?
5. **Agent executability:** Can an agent follow the instructions without ambiguity?
6. **Professional standards:** Does it meet institutional quant finance standards?
7. **Integration:** Does it hand off cleanly to adjacent skills?
